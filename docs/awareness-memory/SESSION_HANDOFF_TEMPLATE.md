# Session Handoff Template

Copy this template into the session record or use it to update `IMPLEMENTATION_STATUS.md`. Do not report work or tests that did not occur.

```text
Session goal:
Current phase:
Bounded task completed:
Files added:
Files modified:
Migrations added:
Decisions made:
Assumptions confirmed or changed:
Tests run:
Tests passed:
Tests failed:
Commands not run:
Known limitations:
Security implications:
Deployment implications:
Unresolved questions:
Current repository state:
Next permitted task:
Required reading for next session:
Explicit stop point:
```

The next permitted task must remain inside the current phase unless the owner has reviewed completion and explicitly authorized the next phase.
